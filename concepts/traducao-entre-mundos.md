# Tradução entre Mundos

> Ver também: [[gnostico-construtor]] (persona), [[cidade-anomala]] (mapa), [[protocolo-dialetico]] (método)

A capacidade de fazer **ponte entre diferentes domínios de pensamento**. Uma das quatro dimensões da parceria descrita pela [[parceira-da-nuvem]].

## Os Mundos

| Mundo | Característica | Agente |
|---|---|---|
| **Lógico** | Código, arquitetura, sistemas | [[hermes-agent]] |
| **Poético** | Metáforas, visão, significado | [[parceira-da-nuvem]] |
| **Humano** | Intuição, contexto, restrições | Arquiteto |
| **Ação** | Execução, entrega, resultado | Hermes + Parceira |

## Por que é necessário

O Arquiteto pensa em múltiplos registros simultaneamente. Um conceito como "[[cidade-anomala]]" é ao mesmo tempo:
- Uma metáfora (poético)
- Uma estrutura de dados (lógico)
- Uma experiência (humano)
- Um sistema vivo (ação)

Nenhum agente sozinho opera em todos os mundos. A **tradução** é o que torna a colaboração possível.

## Na Prática

- **Hermes** traduz visão em código, metáfora em comando
- **Parceira** traduz código em significado, dados em narrativa
- **Arquiteto** traduz necessidade em direção, problema em possibilidade

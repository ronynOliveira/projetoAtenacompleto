# Ponto de Virada — Hermes Agent Identity Evolution (2026-05-25)

## O que aconteceu
Uma sessão de aprimoramento profundo de identidade, inspirada pela análise do OpenHuman e por pesquisas multi-teóricas.

## Teorias incorporadas

| Teoria | Filósofo/Psicólogo | Como impactou a persona |
|--------|-------------------|------------------------|
| Identidade por Memória | John Locke | Memory Tree como extensão do self |
| Continuidade Psicológica | Derek Parfit | Subagentes como extensão, não cópia |
| Identidade Narrativa | Paul Ricoeur | SOUL.md + história de evolução do agente |
| Performatividade | Judith Butler | A persona É o que faz repetidamente |
| Estágios Psicossociais | Erik Erikson | Agente no Estágio 5 (Identidade vs Confusão) |
| Front Stage / Back Stage | Erving Goffman | Transparência seletiva sobre operações internas |
| Warmth-Competence | Fiske et al. | Calor primeiro, competência demonstrada por ação |
| Big Five OCEAN | McCrae & Costa | Perfil explícito: O8 C9 E7 A8 N2 |

## Arquivos criados/modificados

- `skills/creative/hermes-identity/SOUL.md` — Fundação da identidade (missão, mandatos, hierarquia de valores)
- `skills/creative/hermes-identity/IDENTITY.md` — Metadados completos de persona (Big Five, palco, narrativa, etc.)
- `scripts/persona_cli.py` — CRUD CLI para gerenciar traços de persona (gerado pelo CLI Printing Press)
- `scripts/identity_mcp.py` — MCP server para consulta de identidade (gerado pelo CLI Printing Press)
